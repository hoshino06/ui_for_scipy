integrate_solve_ipv モジュール¶

関数 scipy.integrate.solve_ivp() に対するAPIを提供するモジュールです. 初期値問題を解く場合には, InitialValueProb を利用してください. また, integrate.solve_ivpに与える関数のみを得る場合には Vectorfield を利用してください.
